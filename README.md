# Приложение vk shop для отбора на стажировку VK
## Реализованы:  
Карточки товаров(с возможностью перехода на них)  
Кэш данных  
Локальный поиск(по закэшированным данным)  
Поиск по интернету  
Прописаны дополнительные поля на карточках  
Светлая и темная тема
